CSS OPACITY PROPERTY

The CSS opacity property specifies the opacity/transparency of an element.
It takes a value from 0.0 - 1.0.

NOTE: The lower the value, the more transparent it becomes.
opacity 1 is the default, no-transparency value
opacity 0.5 is the medium transparency value,
and the 0.0 is the absolute transparency value

B. OPACITY PROPERTY: 0.5

C. OPACITY PROPERTY: 0.0
