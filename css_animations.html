CSS ANIMATIONS

An animation allows an element to gradually change from one style to another style. With CSS animations, you can change as many CSS properties as you wish to, as many times as you wish.

The CSS animation properties are supported by most browsers like Google Chrome, Internet Explorer, Opera MIni, Safari and Mozilla Firefox. However it depends on the particualr version of the browser.

    These are the CSS animation properties:
  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
