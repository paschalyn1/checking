CSS LINKS

Using CSS one can style links in many diffrent ways.
You can style links as text with background-color, text with an underline, texts with borders or texts with hover effect.

Also, links can also be styled differently depending on what state they are in.
The four states can be: normal/unvisited link, visited link, hovered link and active link
Note: When styling links the style for 'unvisited' comes first followed by the 'visited' link style and then the 'hover' before the 'active'.
