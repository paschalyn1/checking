CSS PSEUDO-CLASSES

The CSS PSEUDO-CLASS is a keyword added to a selector, which is used to define a special state of the selected element(s).

The syntax is written as:
slector:pseudo-class{
property: value;
}

There are different pseudo classes in CSS:
i. :hover
ii. :first-child
iii. :last-child
iv. :nth-child().
v. :focus
