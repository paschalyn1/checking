CSS TRANSFORMS

CSS tranforms allow you to move, rotate, skew and scale elements with the CSS transform property.

This is a 3D transformation.

The CSS transform property are supported by most browsers like Google Chrome, Internet Explorer, Opera MIni, Safari and Mozilla Firefox. However it depends on the particualr version of the browser.

    These are the 2D transformation methods for the CSS transform property:
  • translate()
  • rotate()
  • scale()
  • scaleX()
  • scaleY()
  • skewX()
  • skewY()
  • skew()
  • Matrix()
    These are the 3D transformation methods for the CSS transform property:
  • rotateX()
  • rotateY()
  • rotateZ()
