CSS GRID LAYOUT

CSS Grid layout provides a two dimensional layout system, controlling layout in rows and columns.

Note: The CSS grid properties are supported in all mordern browsers.

Grid properties for the grid container are:display, grid-template-columns, grid-template-rows, justify-content and align-items, gap(column & row gap), height
Grid properties for the grid items in the container are: grid-column, grid-row, grid-column-start, grid-column-end, grid-row-start, grid-row-end, grid-area

Note: To place an item, you can refer to line numbers, or use the keyword "span" to define how many columns the item will span.
