CSS PSEUDO-ELEMENTS

The CSS PSEUDO-ELEMENT is used to style specified parts of an element.

The syntax is written as:
selector::pseudo-element{
property: value;
}

There are different pseudo elements in CSS:
i. ::first-line
ii. ::first-letter
iii. ::before
iv. ::after.
v. :focus
vi. ::marker
vii. ::selection
