CSS OVERFLOW PROPERTY

The CSS overflow property is used to specify what happens to a content if it is too big or too long to fit into the element's box.
It specifes whether a content is to be clipped or should contain scrollbars, when the content is too large to fit in the specified area.

NOTE:This property only works for block elements with a specified height.

It has 4 values: auto, visible, hidden and scroll.
