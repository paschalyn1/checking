CSS COMBINATORS

The CSS combinators are used to explain the relationship between the selectors.
A CSS selector can contain more than one simple selectors, and inbetween these selectors, a CSS combinator can be included.
There are 4 CSS combinators:
the descendant selector (this means adding a space between the selectors),
the child selector (adds > between the selectors),
adjacent sibling selector (adds + between the selectors),
general sibling selector (adds ~ between the selectors)
